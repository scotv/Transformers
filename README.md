Transformers
============

## Build Env

1. install node

2. install yo 

  `npm install -g yo`

3. install nodemon

  `npm install -g nodemon`

2. cd into project directory

3. `npm install`

4. change default to https

  `git config --global url."https://".insteadOf git://`

5. `bower install`

6. `nodemon app.js`
